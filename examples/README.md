# Project Examples

This directory contains example configurations and usage patterns for the Fortran Analyzer framework.

## Available Examples

Each example demonstrates different use cases and project types that the analyzer can handle.

### Running Examples

To run any example:

```bash
cd examples/[example-name]
python run_analysis.py
```

### Creating Custom Examples

You can create your own examples by:

1. Creating a new directory in this folder
2. Adding sample Fortran code
3. Creating a configuration file or using templates
4. Adding a run script to demonstrate the analysis

For more information, see the main documentation in the `docs/` directory.
